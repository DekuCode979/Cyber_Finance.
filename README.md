# 🚀 CyberFinance 2077

![CyberFinance](https://img.shields.io/badge/CyberFinance-2077-ff2d95?style=for-the-badge)
![JavaScript](https://img.shields.io/badge/JavaScript-Vanilla-yellow?style=for-the-badge)
![CSS3](https://img.shields.io/badge/CSS3-Cyberpunk-blue?style=for-the-badge)

Aplicación financiera interactiva con diseño cyberpunk que muestra tasas de cambio en tiempo real de 12 monedas contra el Peso Dominicano (DOP).

## ✨ Características

- 💱 **12 Monedas** con banderas oficiales (USD, EUR, GBP, JPY, CHF, CAD, CNY, MXN, BRL, ARS, CLP, COP)
- 📊 **Gráficos interactivos** con ApexCharts (historial 6 meses)
- 🧮 **Calculador Maestro** de conversión en tiempo real
- 🎨 **Diseño Cyberpunk** con efectos neón y glassmorphism
- 📱 **100% Responsive** (móvil, tablet, escritorio)
- 🔄 **Actualización automática** cada 5 minutos

## 🛠️ Tecnologías

- HTML5 + CSS3 + JavaScript (Vanilla)
- [ExchangeRate-API](https://www.exchangerate-api.com/)
- [ApexCharts](https://apexcharts.com/)
- [Font Awesome](https://fontawesome.com/)
- [FlagCDN](https://flagcdn.com/)
